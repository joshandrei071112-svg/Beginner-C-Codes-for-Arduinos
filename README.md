C++ Beginner Guide for Arduino 

Description
  This project is a beginner-friendly guide in learning C++ programming together with Arduino Development. 
  It includes every basic learning, decision making, loops, functions, arrays, digital I/O and analog I/O.
  Lastly, it contains an example, explanations, and basic programs to help students and even people who wants to learn C++ works in embedded system.

Objectives
- Learn the fundamental of C++
- Understand how C++ is used or works in Arduino
- Practice writing simple programs
- build a strong foundation for advanced projects

Topics covered
- C++
- Variable
- Conditional Statements (if, else, switch)
- Loops (for, while, do-while)
- Arrays
- Functions

Arduino Concept
- Digital I/O
- Analog I/O
- Basic System
- Advanced System

Installation 
1. Download or clone this repository:
https://github.com/joshandrei071112-svg/Beginner-C-Codes-for-Arduinos.git
2. Open the project in Arduino IDE
3. Connect your Arduino board and select the correct port

Usage
- open any .cpp file from the examples
- Upload the code to your Arduino board
- Observe the output using LEDs(or other components being used) or Serial Monitor

Learning Outcome 
After completing each fundamentals of the projects, you will;
- Understand basic C++ syntax
- Be able to write simple Arduino programs
- Know how to control basic electronic components
